Historia

Java desarrollado originalmente por James Gosling, de Sun Microsystems (constituida en 1983 y posteriormente adquirida por la compañia Oracle en 2010.

Es un lenguaje de programación y una plataforma informática comercializada en 1995 por Sun Microsystems.Gracias a Java funcionan muchos sitios web sin el no seria posible, Java es rápido, seguro y confiable.

Desde ordenadores portátiles hasta centros de datos, desde consola para juegos hasta computadoras avanzadas, desde teléfonos móviles hasta internet.

Java desde 2012, uno de los lenguajes de programación mas populares, en particular para aplicaciónes de clienete servidor de web.

Su sintaxis deriva en gran parte de C y C++ tiene menos utilidades de bajo nivel que cualquiera de ellos.Las aplicaciónes de Java son compiladas a bytecode (clase Jav), que puede ejecutaese en cualquier máquina virtual Java (JVM) sin importar la arquitectura de la computadora subyacente.

Introduccion

Su lema (Escribelo una vez, ejecútalo en cualquier lado), proporcionando un lenguaje independiente de la plataforma y un entorno de ejecición (la JVM) ligero y gratuito para las plataformas más populares, de forma que los binarios (bytecode) se ejecutaran en cualquier plataforma.

El entorno de ejecución era relativamente seguro, los principales navegadores web incorporaron la posibilidad de ejecutar applets Java incrustadas en la página web

Evolución

Desde J2SE 1.4, la evolución ha sido regulada por el JCPa (Java Community Process), que usa Java Specification Request (JSRs) para proponer cambios y especificar camnbios en la plataforma Java.

El lenguje en si mismo está espesificado en la Java Language Specification(JLS), o Especificación del Lenguaje Java.

Los cambios en los JLS son gestionados en JSR 901

  • JDK 1.0 (23 de enero de 1996) -Primer lanzamiento: Oracle Hardware
  • JDK 1.1 (19 de febrero de 1997) -Principales adiciones incluidas:
    • una reestructuración intensiva del modelo de evento AWT (Abstract Windowing Toolkit)
    • clases internas (inner classes)
    • JavaBeans
    • JDBC (Java Database Connectivity), para la integración de base de datos
    • RMI (Remote Method Invocation)
  • J2SE 1.2 (8 de diciembre de 1998) Esta y las siguientes versiones fueron denominadas Java 2 Otras mejoras añadidas incluian:
    • la palabra reservada (keyword) strictfp
    • reflexión en la programación
    • la api gráfica (Swing) fue integrada en las clases básicas
  • J2SE 1.3 (8 de mayo de 2000) la inclusión de la máquina virtual de HotSpot
  • J2SE 1.4 (6 de febrero de 2002) Comunidad de Java JSR 59
  • JSE 5.0 (30 de setiembre de 2004)
    • Bucle for mejorado -La sintaxis para el bucle for se ha extendido con una sintaxis especial para iterar sobre cada miembro de un array o sobre cualquier clase que implemente iterable, como la clase estandar Collection, de la siguiente forma: 
                            void displayWidgets (Iterable widgets) {
                            for (Widget w : widgets) {
                                w.display();
                            }
                        }
  • Java SE 6 (11 de diciembre de 2006) En esta versión Sun cambió el nombre "JS2E" por Java SE y se elimino el "0" del número de versión.
  • Java SE 7 Su lanzamiento julio de 2012
  • Java SE 8 Lanzamiento marzo de 2014
  • Java SE 9 -lanzada el 21 de setiembre de 2017
  • Java SE 10 -lanzada el 20 de marzo de 2018
  • Java SE 11-lanzada el 25 de setiembre de 2019
  • Java SE 12 -lanzada el 19 de marzo de 2019
  • Java SE 15 -lanzada el 15 de setiembre de 2020

Filosofia

El lenguaje de Java se creo con cinco objetivos prinscipales:

  1. Debería usar el paradigma de la programación orientada a objetos.
  2. Debería permitir la ejecución de un mismo programa en multiple sistemas operativos.
  3. Debería incluir por defecto soporte para travajo en red.
  4. Debería diseñarse para ejecutar código en sistemas remotos de forma segura.
  5. Debería ser fácil de usar y tomar lo mejor de otros lenguajes orietado a objetos, como C++.
